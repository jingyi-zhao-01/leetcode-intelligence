# LeetCode Submissions

This directory contains LeetCode problem solutions and templates.

## Problems Included

### Easy Problems
- **001_two_sum.py** - Two Sum
- **020_valid_parentheses.py** - Valid Parentheses  
- **121_best_time_to_buy_sell_stock.py** - Best Time to Buy and Sell Stock
- **206_reverse_linked_list.py** - Reverse Linked List

### Medium Problems
- **003_longest_substring_without_repeating.py** - Longest Substring Without Repeating Characters

## Usage

Each file contains:
- Problem description and examples
- Solution class with method signature
- Test cases to verify your implementation

To run a specific problem:
```bash
poetry run python submissions/001_two_sum.py
```

## Implementation Guidelines

1. Implement the solution in the respective method of the `Solution` class
2. Test your solution using the provided test cases
3. Consider edge cases and constraints mentioned in the problem description
4. Optimize for both time and space complexity where possible
